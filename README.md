# CMOR_Capstone_2025-2026
This is the repository for Vikram Shah's, E.K. Turner's, and Kathryn Hu's CMOR 492/493 capstone project for the 2025-2026 academic year.

This project is conducted with the aid of Dr. Sebastian Acosta, PhD and Fabio Savorgnan, MD.

This repository contain the source code for a clinical tool to model Norwood Circulation.

# Repo Structure
The `.ipynb` files in the **CMOR_Capstone_2025-2026** repo are examples notebooks for how the solver work.

The **src/** directory contains the files needed to run the Noorwood circulation web-based fluid solver.

# Running the Web Tool Locally
Follow the steps below to run the web tool on your computer.
## Instructions for running the web tool
1. Navigate to the local copy of the repository on your computer.
2. Activate the virtual environment by running `source ./venv/bin/activate`
3. Start the web tool locally by running `python src/app.py`
