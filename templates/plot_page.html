Variable Response Plots

Our model of the hemodynamics of a Norwood circulatory system solves for 8 variables using 10 parameters.

In this page, you can select any one of the 8 variables, to generate a 2D plot. The plot will display how the selected variable changes (on the y-axis) as each of the 13 parameters changes (on the x-axis). The x-axis is normalized to represent each parameter as it goes from 0.5x to 1.5x its baseline value.

Note: not all variable response plots will show all parameters as not all parameters are relevant for each variable.
